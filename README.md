building & testing
-----------

install `bower` and `gulp` (if not already installed) :
    
    npm install bower -g
    npm install gulp -g

clone git repo :

    git clone git@github.com:risq/cratedigger.js.git cratedigger.js
    cd cratedigger.js

install dependencies :
    
    npm install
    bower install
    
test cratedigger.js :
    
    gulp
    
build cratedigger.js :

    gulp build
    
enjoy !!
